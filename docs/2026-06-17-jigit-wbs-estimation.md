## Quy đổi

- **≈ 53.5 person-days** ≈ **~11 tuần làm việc** (1 dev full-time, 5 ngày/tuần) ≈ **2.5–3 tháng lịch** kể cả buffer debug/review.
- Nếu chia 2 dev song song (1 backend/worker, 1 dashboard/API) theo dependency, có thể rút về **~6–7 tuần lịch**.

## Điểm đáng chú ý

- **3 mục rủi ro cao nhất** chiếm gần 1/3 tổng effort: ACP client (protocol JSON-RPC qua stdio, tài liệu mỏng), LangGraph StateGraph + checkpointer tùy biến, và built-in MCP server — đây là phần một dev thủ công sẽ tốn nhiều thời gian dò/đọc source thư viện nhất.
- Các bug fix "ACP timeout/pause" và "ACP stop reject pending" (11.5, 11.6) là race-condition/deadlock — loại bug thường tốn nhiều giờ debug thực tế hơn số dòng code sửa.
- Đối chiếu: toàn bộ 98 commits này được thực hiện trong **~3–4 ngày lịch** (14–17/06/2026) với AI pair-coding — tức là **~13–17×** nhanh hơn so với ước tính code tay ở trên.
