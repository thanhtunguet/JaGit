# 2026-06-23 14:00 Format Tokens

## Task
Consolidate the behavior of displaying the number of tokens in the Dashboard-V2's Overview and Usage pages, applying the following rules:
- Suffixes: `k`, `M`, `B` for `10^3`, `10^6`, `10^9`.
- Round them and control the decimal places based on the integer part:
  - 1 digit -> max 3 decimal digits
  - 2 digits -> max 2 decimal digits
  - 3 digits -> max 1 decimal digit
