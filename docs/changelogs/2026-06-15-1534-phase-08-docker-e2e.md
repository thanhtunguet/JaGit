## Issues resolved during implementation
1. **SpaController `@All("*path")`** — Fastify rejects wildcard not at end; switched to `@All("*")`
   then hit duplicate OPTIONS registration (NestJS CORS + SpaController both register `/*`).
   Resolution: removed SpaController entirely; @fastify/static handles static assets directly.
2. **`JIGIT_FAKE_ADAPTERS` mapping bypass** — worker threw "No repo mapping" because the E2E
   webhook used project key "E2E" with no seeded mapping. Added fake mapping object in the
   useFakeAdapters branch so the check is bypassed in fake mode.
