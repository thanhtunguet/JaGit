# Chart Colors Update (2026-06-21)

## Summary
Updated the CodeBurn Usage dashboard charts to use a distinctive color palette instead of a single uniform color.

## Changes Made
- Created `packages/dashboard/src/lib/colors.ts` containing `CHART_COLORS`, a palette of 10 distinct colors.
- Modified the following usage components to use `Cell` from `recharts` mapping to `CHART_COLORS` so that different models, projects, activities, tools, and commands each get a distinct color:
  - `ModelsChart.tsx`
  - `ProjectsChart.tsx`
  - `ActivityChart.tsx`
  - `ToolsChart.tsx`
  - `ShellCommandsChart.tsx`

## Testing
- Ran `@jagit/dashboard typecheck` which passed successfully.
- Hot module replacement (HMR) applied changes cleanly to the local development server.
