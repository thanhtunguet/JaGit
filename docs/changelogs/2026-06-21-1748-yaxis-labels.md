# Fix Chart YAxis Labels (2026-06-21)

## Summary
Fixed an issue in the Usage Dashboard where Recharts was hiding some categorical Y-axis labels to prevent overlap.

## Changes Made
- Added `interval={0}` to the `<YAxis>` component in the following vertical bar charts to force all category labels to render regardless of available vertical space:
  - `ActivityChart.tsx`
  - `ModelsChart.tsx`
  - `ProjectsChart.tsx`
  - `ToolsChart.tsx`
  - `ShellCommandsChart.tsx`

## Testing
- Verified successful typecheck for `@jagit/dashboard`.
