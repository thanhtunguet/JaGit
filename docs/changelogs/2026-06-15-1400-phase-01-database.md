## Key Adaptation: Prisma 7

The plan was written for Prisma 5/6. Prisma 7 (installed: 7.8.0) introduced breaking changes:
- `url` no longer allowed in `datasource` block of schema — moved to `prisma.config.ts`
- Client must be instantiated with a driver adapter (`PrismaPg`) instead of direct URL
- Default generator output now writes into the pnpm-store's `@prisma/client` package (no `output` field needed)
- `pnpm-workspace` requires `public-hoist-pattern[]=*prisma*` in `.npmrc`
