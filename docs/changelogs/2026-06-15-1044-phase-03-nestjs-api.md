- Build: `pnpm --filter @jigit/api build` — no errors

## Key Decisions

- Used `unplugin-swc` + Vitest config to enable `emitDecoratorMetadata` (esbuild doesn't support it)
- Used `@Res({ passthrough: true })` to override 202→200 for ignored/duplicate webhook responses
- Test helper pattern: explicit provider declarations in TestingModule rather than `overrideProvider` (avoids global scope issues)
- `bullmq` added as direct dependency to `@jigit/api` (needed for `Queue` type in service)
